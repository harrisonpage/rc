# random color

i've been dragging "rc" around from mac to mac for decades.

rc2 is an upgrade:

* 'rc2 preview' generates preview.html 
* 'rc2 green-sea' applies a named color

## shortcuts

```bash
./rc2 preview && open preview.html
```

## historical shit

* `rc` is the original perl script
* `color` is another perl script

features from both have been merged into rc2.
